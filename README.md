## 📋 نظرة عامة

**HAINON** هو نظام محاسبة وإدارة مالية سحابي متكامل، يعمل بواجهة أمامية خالصة (Frontend-only) مرتبطة بـ Firebase. يتميز بواجهة مستخدم عصرية فاخرة، دعم كامل للغة العربية (RTL)، وضعان داكن وفاتح، ونظام VIP متعدد المستويات مع تأثيرات بصرية مميزة.

---

## 🎯 الميزات الرئيسية

### 🔐 المصادقة والأمان
| الميزة | الوصف |
|--------|-------|
| **تسجيل الدخول / إنشاء حساب** | بريد إلكتروني + كلمة مرور |
| **تأكيد البريد الإلكتروني** | نظام OTP متكامل عبر EmailJS |
| **نسيت كلمة المرور** | استعادة عبر رمز تحقق يصل للبريد |
| **سياسة كلمات المرور** | حرف إنجليزي + أرقام، 6 خانات كحد أدنى |

### 💰 العمليات المالية
- **10 بطاقات مالية تفاعلية:** وارد، صادر، بيع، شراء، دين لنا، دين علينا، دين مقبوض، دين مدفوع، مرتجع (مستطيل)، إجمالي الرصيد (مستطيل)
- **نموذج إدخال متقدم:** ديناميكي يتكيف مع نوع العملية
- **مصادقة البيع:** يمنع البيع إذا تجاوز الكمية المتاحة
- **أرشفة تلقائية:** عمليات اليوم السابق تؤرشَف تلقائياً
- **سجل تعديلات:** تتبع كامل لتاريخ التعديل مع أسهم تنقل بين القيم
- **حساب الأرباح والخسائر:** باستخدام طريقة متوسط التكلفة المرجح (VIP 2 فأعلى)

### 👑 نظام VIP الهرمي (5 مستويات)
| المستوى | الميزات |
|---------|---------|
| **VIP 1** | إطار بني، ID مميز، خدمة عملاء |
| **VIP 2** | إطار أخضر، تقارير أرباح/خسائر |
| **VIP 3** | إطار بنفسجي، إزالة العلامة المائية |
| **VIP 4** | إطار أزرق ملكي (للمشرفين تلقائياً) |
| **VIP 5** | إطار ذهبي ناري (للمدير تلقائياً) |

### 🎨 تجربة المستخدم
- **وضعان:** داكن وفاتح مع زر تبديل في الشريط العلوي
- **أيقونات Font Awesome** الأنيقة
- **قائمة جانبية ديناميكية** تتكيف حسب دور المستخدم
- **آلة حاسبة عائمة** قابلة للسحب بحرية كاملة
- **شريط VIP علوي** يعرض رسائل المستخدمين المميزين
- **تأثير رقاقات ذهبية** عند الترقية
- **تنسيق التاريخ بالأرقام الأجنبية** (MM/DD/YYYY)

### 📞 التواصل المحدود والهادف
- **محادثة الإدارة الداخلية:** دردشة خاصة بين الأدمن والمشرفين فقط
- **خدمة عملاء VIP:** محادثة مباشرة مع الإدارة لمستخدمي VIP حصراً، مع لوحة تحكم للمشرفين لعرض جميع جهات الاتصال
- **طلبات ترقية VIP:** نظام تذاكر منظم لمراجعة طلبات الترقية وقبولها/رفضها

### 👥 إدارة المستخدمين (للأدمن والمشرف)
- **جدول كامل:** صورة، اسم، ID، بريد، دور، حالة
- **حقل بحث:** بحث بالاسم أو ID أو البريد
- **مراقبة الجلسات:** Online/Offline، آخر ظهور، IP، موقع، جهاز
- **إجراءات:** تعيين مشرف/مشرف مميز، تعيين VIP، تعديل ID، حظر، حذف
- **قسم طلبات VIP المعلقة** مع زري قبول/رفض
- **إرسال إشعار جماعي:** زر في الشريط السفلي لإرسال إشعار لجميع المستخدمين

### 📊 التقارير
- **تقرير مخصص** حسب الفترة ونوع العملية
- **علامة مائية** قابلة للتخصيص حسب مستوى VIP
- **حساب الأرباح والخسائر** (لمستخدمي VIP 2 فأعلى)

### 🏅 نظام مستوى الحساب (LV)
- **LV 0:** المستخدم الجديد
- **LV 1:** بعد 2500 عملية
- **LV 2:** بعد 4000 عملية
- يظهر المستوى بجانب اسم المستخدم في كل مكان
- صفحة مخصصة لتتبع التقدم

### ⚙️ الإعدادات
- **تعديل الملف الشخصي:** اسم، صورة، غلاف، سيرة ذاتية
- **تغيير كلمة المرور:** بدون الحاجة لكلمة المرور القديمة
- **قص الصورة** عبر Cropper.js

### ✍️ شريط الكتابة
- متاح لـ VIP والمشرفين والمدير
- نص بحد أقصى 90 حرفاً
- اختيار لون الخط من 10 ألوان
- يظهر في الشريط العلوي المتحرك

## 🚀 انطلاق سريع

1. **الاستضافة:** ارفع جميع الملفات إلى استضافة تدعم الملفات الثابتة (Firebase Hosting، GitHub Pages...).
2. **Firebase:** تأكد من تفعيل خدمات Firestore و Realtime Database.
3. **قواعد Firestore:** اضبط القواعد للسماح بالقراءة والكتابة للمستخدمين المسجلين فقط.
4. **EmailJS:** تأكد من صحة المفاتيح في `firebase-config.js`.
5. **أول مستخدم:** أول حساب يتم إنشاؤه يجب تعيينه كـ Admin يدوياً من لوحة تحكم Firebase (تعديل حقل `role` إلى `admin`).
