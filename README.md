## 📋 نظرة عامة

**HAINON** هو نظام محاسبة وإدارة مالية سحابي متكامل، يعمل بواجهة أمامية خالصة (Frontend-only) مرتبطة بـ Firebase. يتميز بواجهة مستخدم عصرية فاخرة، دعم كامل للغة العربية (RTL)، وضعان داكن وفاتح، نظام VIP متعدد المستويات، وبطاقات ديناميكية تفاعلية.

---

## 🎯 الميزات الرئيسية

### 🔐 المصادقة والأمان
| الميزة | الوصف |
|--------|-------|
| **تسجيل الدخول / إنشاء حساب** | بريد إلكتروني + كلمة مرور + اسم الشركة |
| **تأكيد البريد الإلكتروني** | نظام OTP متكامل عبر EmailJS |
| **نسيت كلمة المرور** | استعادة عبر رمز تحقق يصل للبريد |
| **سياسة كلمات المرور** | حرف إنجليزي + أرقام، 6 خانات كحد أدنى |

### 💰 العمليات المالية
- **8 بطاقات ديناميكية:** 7 أزواج قابلة للتبديل + بطاقة ثابتة
- **أزواج البطاقات:** بيع↔شراء، وارد↔صادر، دين علينا↔دين لنا، دين مدفوع↔دين مقبوض، أرباح↔أرباح نهائية، خسائر↔مرتجع، إجمالي الرصيد↔الرصيد النهائي
- **قيمة البضائع:** بطاقة ثابتة تعرض قيمة المخزون المتبقي
- **نموذج إدخال متقدم:** ديناميكي يتكيف مع نوع العملية
- **حقل سعر القطعة الصافي:** في عمليات الشراء
- **حقل السعر النهائي:** في عمليات البيع
- **منسدلة المنتجات:** في البيع تظهر المنتجات المشتراة فقط
- **حساب الأرباح والخسائر تلقائياً:** عند البيع
- **منع العمليات تحت الصفر:** رسالة "إجمالي الرصيد غير كافي"
- **أرشفة تلقائية:** عمليات اليوم السابق تؤرشَف تلقائياً

### 👑 نظام VIP الهرمي (5 مستويات)
| المستوى | الميزات |
|---------|---------|
| **VIP 1** | إطار بني، ID مميز، خدمة عملاء، شريط كتابة، تغيير الغلاف |
| **VIP 2** | إطار أخضر، تقارير أرباح/خسائر، إظهار/إيقاف الشريط العلوي |
| **VIP 3** | إطار بنفسجي، إزالة العلامة المائية |
| **VIP 4** | إطار أزرق ملكي (للمشرفين تلقائياً) |
| **VIP 5** | إطار ذهبي ناري (للمدير تلقائياً) |

### 🏅 نظام مستوى الحساب (LV)
- **نقاط الخبرة:** 500 نقطة لكل عملية
- **مستويات لا نهائية:** كل مستوى يحتاج ضعف النقاط السابقة
- **LV 1:** 2,500 نقطة | **LV 2:** 5,000 | **LV 3:** 10,000 | ...
- **ألوان متغيرة** لكل مستوى
- **صفحة مخصصة** لتتبع التقدم

### 🎨 تجربة المستخدم
- **وضعان:** داكن وفاتح مع زر تبديل
- **أيقونات Font Awesome**
- **قائمة جانبية ديناميكية** تتكيف حسب دور المستخدم
- **آلة حاسبة عائمة** قابلة للسحب (الزر فقط)
- **شريط VIP علوي** يعرض رسائل المستخدمين المميزين
- **تأثير رقاقات ذهبية** عند الترقية
- **تنسيق التاريخ بالأرقام الأجنبية** (MM/DD/YYYY)
- **اسم الشركة** يظهر في الشريط العلوي

### 📞 التواصل المحدود والهادف
- **محادثة الإدارة الداخلية:** للأدمن والمشرفين فقط
- **خدمة عملاء VIP:** للمستخدمين VIP مع لوحة تحكم للمشرفين
- **طلبات ترقية VIP:** نظام تذاكر منظم

### 👥 إدارة المستخدمين
- **جدول كامل:** صورة، اسم، ID، بريد، دور، حالة
- **حقل بحث:** بالاسم أو ID أو البريد أو اسم الشركة
- **إجراءات:** تعيين مشرف، تعيين VIP، تعديل ID، حذف صورة، حظر، حذف
- **قسم طلبات VIP المعلقة**

### 🛠️ إدارة الموقع (للأدمن والمشرفين)
- **إرسال إشعار جماعي**
- **إزالة شريط:** بحث عن مستخدم وحذف شرائطه
- **تعديل طريقة الدفع:** للأدمن فقط مع تعديل الأسعار

### ✍️ شريط الكتابة
- متاح لـ VIP والمشرفين والمدير
- نص بحد أقصى 90 حرفاً مع 10 ألوان
- تعديل الشريط بعد رفعه
- حد زمني: شريط واحد كل 42 ساعة

### ⚙️ الإعدادات
- تعديل الملف الشخصي (اسم، صورة، غلاف، سيرة ذاتية)
- تغيير كلمة المرور (بدون الحاجة لكلمة المرور القديمة)
- إظهار/إيقاف الشريط العلوي (لـ VIP2+)
- قص الصورة عبر Cropper.js

## 🚀 انطلاق سريع

1. **الاستضافة:** ارفع جميع الملفات إلى استضافة ثابتة (Firebase Hosting، GitHub Pages...).
2. **Firebase:** تأكد من تفعيل Firestore و Realtime Database.
3. **قواعد Firestore:** اضبط القواعد للسماح بالقراءة والكتابة للمستخدمين المسجلين.
4. **EmailJS:** تأكد من صحة المفاتيح في `firebase-config.js`.
5. **أول مستخدم:** عيّن أول حساب كـ Admin من لوحة Firebase (حقل `role` = `admin`).
