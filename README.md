## 📋 نظرة عامة

**HAINON** هو نظام محاسبة وإدارة مالية سحابي متكامل، يعمل بواجهة أمامية خالصة (Frontend-only) مرتبطة بـ Firebase. يتميز بواجهة مستخدم عصرية فاخرة، دعم كامل للغة العربية (RTL)، وضعان داكن وفاتح، ونظام VIP متعدد المستويات.

---

## 🎯 الميزات الرئيسية

### 🔐 المصادقة والأمان
| الميزة | الوصف |
|--------|-------|
| **تسجيل الدخول / إنشاء حساب** | بريد إلكتروني + كلمة مرور |
| **تأكيد البريد الإلكتروني** | نظام OTP متكامل عبر EmailJS |
| **نسيت كلمة المرور** | استعادة عبر رمز تحقق يصل للبريد |
| **سياسة كلمات المرور** | حرف إنجليزي + أرقام، 6 خانات كحد أدنى |

### 💰 العمليات المالية
- **10 بطاقات مالية تفاعلية:** وارد، صادر، بيع، شراء، دين لنا، دين علينا، دين مقبوض، دين مدفوع، مرتجع، إجمالي الرصيد
- **نموذج إدخال متقدم:** ديناميكي يتكيف مع نوع العملية
- **مصادقة البيع:** يمنع البيع إذا تجاوز الكمية المتاحة
- **أرشفة تلقائية:** عمليات اليوم السابق تؤرشَف تلقائياً
- **سجل تعديلات:** تتبع كامل لتاريخ التعديل مع أسهم تنقل بين القيم

### 👑 نظام VIP الهرمي
| المستوى | الميزات |
|---------|---------|
| **VIP 1** | إطار ذهبي، ID مميز، خدمة عملاء |
| **VIP 2** | إطار مزدوج، تقارير أرباح/خسائر |
| **VIP 3** | إطار مرصع، تأثيرات فاخرة، إزالة العلامة المائية |
| **مشرف** | مراقبة وحظر |
| **مشرف مميز** | تعيين VIP، تغيير ID مؤقت |
| **أدمن** | صلاحيات كاملة |

### 🎨 تجربة المستخدم
- **وضعان:** داكن وفاتح مع زر تبديل في الشريط العلوي
- **أيقونات Font Awesome** الأنيقة
- **قائمة جانبية ديناميكية** تتكيف حسب دور المستخدم
- **آلة حاسبة عائمة** قابلة للسحب
- **شريط VIP علوي** يعرض رسائل المستخدمين المميزين
- **تأثير رقاقات ذهبية** عند الترقية إلى VIP
- **تنسيق التاريخ بالأرقام الأجنبية** (MM/DD/YYYY)

### 📞 التواصل المحدود والهادف
- **محادثة الإدارة الداخلية:** دردشة خاصة بين الأدمن والمشرفين فقط
- **خدمة عملاء VIP:** محادثة مباشرة مع الإدارة لمستخدمي VIP حصراً
- **طلبات ترقية VIP:** نظام تذاكر منظم لمراجعة طلبات الترقية وقبولها/رفضها

### 👥 إدارة المستخدمين (للأدمن والمشرف)
- **جدول كامل:** صورة، اسم، ID، بريد، دور، حالة
- **مراقبة الجلسات:** Online/Offline، آخر ظهور، IP، موقع، جهاز
- **إجراءات:** تعيين VIP، تعديل ID، إزالة صورة، حظر، حذف
- **قسم طلبات VIP المعلقة** مع زري قبول/رفض

### 📊 التقارير
- **تقرير مخصص** حسب الفترة ونوع العملية
- **علامة مائية** قابلة للتخصيص حسب مستوى VIP
- **حساب الأرباح والخسائر** (لمستخدمي VIP 2 فأعلى) باستخدام متوسط التكلفة المرجح

### ⚙️ الإعدادات
- **تعديل الملف الشخصي:** اسم، بريد، صورة، غلاف، سيرة ذاتية
- **تغيير كلمة المرور**
- **إعدادات الخصوصية:** من يرى ملفي، من يراسلني، إظهار الحالة
- **قص الصورة** عبر Cropper.js

**المجموع:** 7 ملفات (1 HTML + 1 CSS + 5 JS) يمكن وضعها في مجلد واحد أو مع مجلدات فرعية.

## 🚀 انطلاق سريع

1. **الاستضافة:** ارفع جميع الملفات إلى استضافة تدعم الملفات الثابتة (Firebase Hosting، GitHub Pages...).
2. **Firebase:** تأكد من تفعيل خدمات Firestore و Realtime Database في مشروعك على Firebase.
3. **EmailJS:** تأكد من صحة المفاتيح في `firebase-config.js` وأن القالب `template_f7rs16k` يقبل متغيرات `to_email`, `passcode`, `time`.
4. **أول مستخدم:** أول حساب يتم إنشاؤه يجب تعيينه كـ Admin يدوياً من لوحة تحكم Firebase (تعديل حقل `role` إلى `admin`).
