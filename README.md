## 📋 نظرة عامة

**HAINON** هو نظام إدارة وتحليل مالي سحابي متطور، يعمل بالكامل كواجهة أمامية (Frontend-only) مرتبطة بـ Firebase، ومستضافة على GitHub Pages. يتميز بواجهة مستخدم عصرية وفاخرة مع دعم الوضعين الداكن والفاتح، وتجربة عربية متكاملة.

---

## 🎯 الميزات الرئيسية

### 🔐 المصادقة والأمان
| الميزة | الوصف |
|--------|-------|
| **تسجيل الدخول / إنشاء حساب** | بريد إلكتروني + كلمة مرور |
| **تأكيد البريد الإلكتروني** | نظام OTP متكامل عبر EmailJS |
| **نسيت كلمة المرور** | استعادة عبر رمز تحقق يصل للبريد |
| **سياسة كلمات المرور** | حرف إنجليزي + أرقام، 6 خانات كحد أدنى |
| **أول مستخدم = Admin** | الرقم التسلسلي: 11110 |

### 💰 العمليات المالية
- **10 بطاقات مالية تفاعلية:** وارد، صادر، بيع، شراء، دين لنا، دين علينا، دين مقبوض، دين مدفوع، مرتجع، إجمالي الرصيد
- **نموذج إدخال متقدم:** ديناميكي يتكيف مع نوع العملية
- **مصادقة البيع:** يمنع البيع إذا تجاوز الكمية المتاحة
- **أرشفة تلقائية:** عمليات اليوم السابق تؤرشَف تلقائياً
- **سجل تعديلات:** تتبع كامل لتاريخ التعديل مع أسهم تنقل بين القيم

### 👑 نظام VIP الهرمي
| المستوى | الميزات |
|---------|---------|
| **VIP 1** | إطار ذهبي، ID مميز، 50 صديق |
| **VIP 2** | إطار مزدوج، 100 صديق |
| **VIP 3** | إطار مرصع، 300 صديق، تأثيرات فاخرة |
| **مشرف** | مراقبة وحظر |
| **مشرف مميز** | تعيين VIP، تغيير ID مؤقت |
| **أدمن** | صلاحيات كاملة |

### 🎨 تجربة المستخدم
- **وضعان:** داكن وفاتح مع زر تبديل في الشريط العلوي
- **أيقونات Font Awesome** الأنيقة
- **قائمة جانبية ديناميكية** تتكيف حسب دور المستخدم
- **آلة حاسبة عائمة** قابلة للسحب
- **شريط VIP علوي** يعرض رسائل المستخدمين المميزين
- **تأثير رقاقات ذهبية** عند الترقية إلى VIP
- **لون خط قابل للتخصيص** من صفحة الإعدادات

### 💬 التواصل الاجتماعي
- **تكوين صداقات:** حدود مختلفة حسب مستوى VIP
- **مراسلة خاصة** بين الأصدقاء
- **مجموعات:** مع حذف تلقائي للرسائل القديمة
- **خدمة عملاء** للمستخدمين
- **مشاكل المستخدمين** للأدمن والمشرفين مع تصفية

### 👥 إدارة المستخدمين (للأدمن والمشرف)
- **جدول كامل:** صورة، اسم، ID، بريد، دور، حالة
- **مراقبة الجلسات:** Online/Offline، آخر ظهور، IP، موقع، جهاز
- **إجراءات:** تعيين VIP، تعديل ID، إزالة صورة، حظر، حذف
- **مراجعة الصور:** تنبيهات فورية عند رفع صور جديدة

### 📊 التقارير
- **تقرير يومي، أسبوعي، شهري**
- **تصفية متعددة:** مع رقاقات تفعيل لكل نوع عملية
- **علامة مائية** قابلة للتخصيص حسب مستوى VIP
- **تصدير PDF** (قيد التفعيل)

### ⚙️ الإعدادات
- **تعديل الملف الشخصي:** اسم، بريد، صورة، غلاف، سيرة ذاتية
- **تغيير كلمة المرور**
- **إعدادات الخصوصية:** من يرى ملفي، من يراسلني، إظهار الحالة
- **تخصيص لون الخط**
