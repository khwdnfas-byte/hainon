## 📋 نظرة عامة

HAINON هو نظام إدارة وتحليل مالي سحابي متطور، يعمل بالكامل كواجهة أمامية (Frontend-only) مرتبطة بـ Firebase، ومستضاف عبر GitHub Pages.

---

## 🎯 الميزات الرئيسية

### 🔐 المصادقة والأمان
- تسجيل دخول / إنشاء حساب (بريد إلكتروني + كلمة مرور)
- سياسة كلمات مرور قوية (حرف إنجليزي + أرقام، 6 خانات كحد أدنى)
- أول مستخدم يصبح Admin تلقائياً برقم تسلسلي `11110`
- حماية كاملة عبر Firebase Auth

### 💰 العمليات المالية
- **6 أنواع للعمليات:** وارد، صادر، بيع، شراء، دين، مرتجع
- **دعم عملتين:** USD (دولار) و SYP (ليرة سورية)
- **حسابات آلية:**
  - الأرباح والخسائر = (سعر البيع - سعر الشراء) - المرتجع
  - الصافي = (الوارد + البيع) - (الصادر + الشراء)
- الديون مؤرشفة ولا تؤثر على الصافي

### 🎨 تجربة المستخدم
- تصميم داكن (Dark Mode) فاخر
- دعم كامل للغة العربية (RTL)
- قائمة جانبية تفاعلية مع صورة شخصية
- أكورديون لإدخال العمليات
- آلة حاسبة عائمة
- نظام تنبيهات Toast

### 💬 التواصل
- محادثة آنية (Real-time Chat) بين المستخدمين والأدمن
- تحديثات لحظية لجميع البيانات

### 👤 إدارة الملف الشخصي
- صورة شخصية تفاعلية (دائرية)
- إعداد أولي (Onboarding) عند أول تسجيل دخول
- إمكانية تغيير الصورة من الإعدادات
